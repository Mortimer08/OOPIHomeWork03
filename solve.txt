Фигуры:

Классы:

1 Абстрактный класс Shape Фигура имплементирует станартный интерфейс CompareTo, имплементирует интерфейс CalculateArea.
    Поле - наименование фигуры shapeName
    Конструктор по умолчанию.
    Переопределение метода toString() - возвращает наименование фигуры shapeName;
    Переопределение метода compareTo
1.1 Абстрактный класс Polygon Многоугольник имплементирует Интерфейсы вычисление периметра
        (CalculatePerimeter) (и вычисление площади (CalculateArea)),
    Поле - список Integer.
    Конструктор - принимает список чисел передаёт в модификатор.
    Переопределение абстратныого метода вычисления периметра CalculatePerimeter - 
        получает список с длинами сторон, суммирует и возвращает результат.
    Переопределение метода toString() - возвращает результат родительского toString() и
        значения периметра и площади.
    Модификатор - заносит числа-значения длин сторок в список.
1.1.1 Класс Triangle Треугольник
    (Поле - список из трёх чисел (наследует от класса Polygon))
    Переопреление метода вычисления площади calculateArea: 
        S = sqrt(p(p-a)(p-b)(p-c)), где р = (a+b+c)/2
    Конструктор принимает три числа. Передаёт в модификатор.
        Передаёт в родительский конструктор (Polygon) имя "Треугольник" и список чисел.
    Модификатор значений. Получает три числа, проверяет длины сторон - сумма двух чисел должна быть больше третьего (три проверки),
        передаёт в родительский модификатор.
1.1.2 Класс Retrangle Прямоугольник
    (Поле - список из четырёх чисел (наследует от класса Polygon))
    Переопреление метода вычисления площади calculateArea: 
        S = a * b
    Конструктор - получает два числа. (?) Проверяет на равенство.
        Если числа неравны - передаёт в родительский конструктор (Polygon) имя "Прямоугольник" и список чисел.
        (?) Если равны -  передаёт число в конструктор (Square) число.
    Модификатор получает список чисел. Проверяет, что они положительные. Передаёт в родительский модификатор.
1.1.2.1 Класс Square Квадрат
    (Поле - список из четырёх чисел (наследует от класса Retrangle))
    Конструктор - получает одно число. Проверяет, положительное ли оно.
        Передаёт в родительский конструктор (Retrangle) имя "Квадрат" и список чисел.
1.2 Класс Circle Круг имплементирует интерфейс вычисление длины оружности (CalculateCircleLenght)
        и вычисление площади (CalculateArea).
    Конструктор - получает число. Проверяет на положительность. Передаёт в родительский конструктор (Shape)
        наименование "Круг".
    Переопределение метод вычисления длины окружности СalculateCircleLenght
    Переопределение метода вычисления площади calculateArea

Интерфейсы:

1 Интерфейс вычисление длины окружности CalculateCircleLenght.
2 Интерфейс вычисление периметра CalculatePerimeter
3 Интерфейс вычисления площади CalculateArea

Список фигур:

Классы:

1. Класс ShapesCollection имплементирует интерфейс Comparable
    Поле - список экземпляров типа фигура (Shape)
    Конструктор по умолчанию без аргументов - инициализирует список
    Метод добавления экземпляра в коллекцию addShape с аргументом типа Shape
    Метод удаления фигуры из коллекции removeShape с аргументом типа Integer
    Метод вывода инфомации о периметрах всех фигур, для которых можно вычислить периметр
        showPerimeters. Проверяет фигуру на отношение к интерфейсу calculatePerimeter,
        приводит к типу CalculatePerimeter, вычисляет и выводит в консоль
    Метод вывода инфомации о длинах окружности всех фигур, для которых можно вычислить длину окружности
        showCircleLenghts. Проверяет фигуру на отношение к интерфейсу CalculateCircleLenght,
        приводит к типу CalculateCircleLenght, вычисляет и выводит в консоль
    Метод вывода инфомации о площадях всех фигур, для которых можно вычислить площадь
        showCircleLenghts. Проверяет фигуру на отношение к интерфейсу CalculateCircleLenght,
        приводит к типу CalculateCircleLenght, вычисляет и выводит в консоль
    Метод изменения фигуры по индексу. Принимает индекс фигуры и числа для определения параметров фигуры.
        Переназначает значения соответствующей фигуры через модификаторы.
    Метод вывода информации о всех фигурах перебирает все фигуры в коллекции и получает toString() и выводит в консоль.
    Метод сортировки по площади sortBySquare использует станартный метод сортировки элементов коллекции Collection.sort()
